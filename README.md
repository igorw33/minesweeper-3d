
# Minesweeper 3D

An online minesweeper 3D co-op game for 2 players.

## Run Locally

Clone the project

```bash
  git clone https://github.com/igorw33/minesweeper-3d.git
```

Go to the project directory

```bash
  cd [project directory]
```

Install necessary modules
```bash
  npm install
```

If needed, run
```bash
  npm audit fix
```

Start server.js
```bash
  node server
```

Open localhost on port 3000 in web browser of your choice

## Tech Stack

**Client:** JavaScript, Three.js, HTML, CSS

**Server:** Node.js, Express + NoDB database
